PowerBasicObject 模型对象基类

    PowerBasicObject 类为模型对象基类，它是抽象类。

命名空间：WpfEarthLibrary

1. 字段

名称 说 明
submodels 该模型对象的子对象字典集合。
VecLocation 模型对象在三D空间的位置，可用于移动相机以观看此模型对象。
center 中心点经纬度，y经 x纬
bounds 原始坐标表示的范围，若为经纬度，x表示纬度, y表示经度
isHandled 动态载入时使用，是否已处理
latitude 纬度
groundHeight 离地高度
visualMaxDistance 可视的最大距离，缺省100，即相机远端距离，动态显示模式下有效
visualMinDistance 可视最小距离，缺省0，动态显示模式下有效
   

2. 属性

名称 说 明
id 模型对象ID
name 模型对象名称
isShow 此模型是否显示
isShowSubObject 是否显示此模型的子对象
busiData 扩展的业务数据对象
busiAccount 扩展的台账数据对象
Progress 控制绘制进度
Label 对象的内置文字标签
isShowLabel 是否显示内置文字标签，isShowLabel应最后设置，之前设置的标签属性才能生效
LabelPanelKey 内置标签文字面板的纹理键，可不设置。
LabelSizeX 内置标签X缩放系数, 缺省1
LabelSizeY 内置标签Y缩放系数, 缺省1
LabelColor 内置标签文字色
tooltipMoveTemplate 设置mouse move形态的模板名
tooltipMoveContent 设置mouse move形态的内容
tooltipClickTemplate 设置mouse click形态的模板名
tooltipClickContent 设置mouse click形态的内容

3. 方法

名称 说 明
AddSubObject 添加数据对象到submodels。
   

4. 事件

名称 说 明
