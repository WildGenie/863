pSymbolObject 图元类

    pSymbolObject 类绘制图元。

命名空间：WpfEarthLibrary

1. 字段

名称 说 明
isColorManaged 对象是否应用色彩管理
isUseColor 图元是否可使用颜色为材质。若为false，图元将使用symbolid指定的纹理资源本身的颜色。若为true，图元将使用color属性指定的颜色来渲染图元。
aniTwinkle 控制闪烁动画的参数
aniShow 控制淡入淡出显示动画的参数
aniScale 控制缩小放大动画的参数

2. 属性

名称 说 明
location 设置和读取图元的经纬坐标，字符串类型。
planeLocation 设置和读取图元的平面坐标，字符串类型。仅用于平面模式下。
objType 预定义的枚举类型的对象类型，通过设置时自动改变objTypeString生效，与颜色管理配套使用。
objTypeString 对象类型字串，与颜色管理配套使用。
objStatus 预定义的枚举类型的状态，通过设置时自动改变objStatusString生效，与颜色管理配套使用。
objStatusString 状态字串，与颜色管理配套使用。
symbolid 指定图元所使用的纹理资源ID，关于纹理资源参见 pObjectManager Class
color 设置用于渲染图元的颜色，当isUseColor为true时有效。
scaleX 图元的横向缩放比例，缺省0.025。
scaleY 图元的纵向缩放比例，缺省0.025。
scaleZ xmodel的高度缩放比例，缺省0.025。
isH 图元是否水平放置，缺省false。
isUseXModel 是否使用3D模型
XModelKey 3D模型字典中的键值
XMScaleAddition 3D模型时，ScaleX Y X的附加比例，用于匹配使得3D和2D的大小一致。3D的实际计算大小为scaleX * XMScaleAddition
   
   
   
   
   
   

3. 方法

名称 说 明
AnimationBegin 提交开始执行一个动画，动画参数由相应的动画字段AniShow, AniTwinkle, AniScale结构决定
AnimationStop 提交停止一个动画
SendAniPara 提交动画参数，可用于控制动画播放、停止和刷新动画的参数
