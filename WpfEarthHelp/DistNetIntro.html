DistNetLibrary配网类库简单介绍

一、DistNetLibray配网类库基于WpfEarth类库，配网对象为WpfEarth类库中的pSymbol图元类、pPowerline折线类等的派生类，因此，它们的基类均为PowerBasicObject类。

　　配网对象类，扩展使用了PowerBasicObject类的下面四个属性

///<summary>扩展业务对象描述数据，一般为静态数据</summary>
public object busiDesc { get; set; }
///<summary>扩展业务台账数据，一般为静态数据</summary>
public object busiAccount { get; set; }
///<summary>扩展业务运行数据，一般为动态数据</summary>
public object busiRunData { get; set; }
///<summary>扩展业务对象拓扑数据</summary>
public object busiTopo { get; set; }

　　其中，所有的配网对象类的busiDesc描述属性和busiTopo拓扑属性两个属性都有相同的数据结构，而busiAccount台账属性和busiRunData运行属性则各不相同，但它们又都有各自相同的基类AcntDataBase和RunDataBase。

　　上述四个属性在配网对象中可以分别用thisDesc、thisAccount、thisRunData、thisTopoData指代，方法获得他们各自的属性。

二、DistNetLibrary配网类库还提供了一个配网类DistNet，它封装了Earth类，并提供了一些基础属性和方法，如获取对象方法。

此外，DistNet还封装了一些配网功能性的方法，如显示潮流、显示负载、显示电压等值图、电源追溯、显示供电范围、拓扑搜索等，今后根据需要还将增加。
